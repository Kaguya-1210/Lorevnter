# Lorevnter

> AI 驱动的 SillyTavern 世界书自动管理插件

## ✨ 功能概览

Lorevnter 是一款 SillyTavern 酒馆助手脚本插件，通过 AI 自动读取聊天正文、分析剧情变化，并根据你定义的约束规则自动更新世界书条目和用户人设，实现世界观数据的动态维护。

### 🎯 核心特性

- **自动世界书更新** — AI 分析聊天正文，自动提出条目修改/新增建议
- **约束系统** — 为每个条目设定专属 AI 处理规则（提示词约束 / 跳过约束）
- **审核机制** — 所有修改经人工审核确认后才写入，支持 Diff 对比
- **CoT 思考链** — AI 逐条分析每个条目，确保判断精准
- **用户人设同步** — AI 可同步更新角色人设信息
- **宏系统** — 支持条目级短标签 + 酒馆原生宏
- **自动备份** — 分析前自动快照，支持手动回档

## 📦 安装

1. 从本仓库下载 `dist/Lorevnter/Lorevnter.js` 文件
2. 在 SillyTavern 中打开**酒馆助手**面板
3. 点击「导入脚本」，选择下载的 `Lorevnter.js` 导入
4. 顶部工具栏出现 **Lorevnter** 按钮即安装完成

## 🚀 快速上手

### 1. 打开插件

点击顶部工具栏的 **Lorevnter** 按钮，或使用斜杠命令 `/lore`。

### 2. 配置 AI

在 **AI 配置** 页签中：
- **API 来源**：选择「复用酒馆连接」（推荐）或「自定义端点」
- **采样参数**：Temperature / Top P / Max Tokens 可单独设置或跟随酒馆预设
- **触发方式**：自动（每 N 轮 AI 回复后）或手动

### 3. 设置约束

在 **约束管理** 页签中为你的条目设置处理规则：
- **提示词约束**：告诉 AI 如何分析这个条目（如"记录能力值变化"、"自由扩展内容"）
- **跳过约束**：标记不需要 AI 处理的条目
- 支持全局约束和角色卡级局部约束

### 4. 开始分析

- **自动模式**：AI 回复达到设定间隔后自动触发
- **手动模式**：在世界书页签点击「🔍 AI 分析」按钮

### 5. 审核修改

AI 分析完成后弹出审核面板：
- 逐条查看修改建议和理由
- 展开查看行级 Diff 对比
- 通过 / 拒绝后确认写入

## 🛠️ 功能详解

### 约束系统

| 类型 | 作用 | 示例 |
|---|---|---|
| **提示词约束** | 告诉 AI 如何处理条目 | "记录能力值变化，人际关系发展" |
| **跳过约束** | 排除 AI 处理 | 固定设定类条目 |

约束支持全局（跨角色卡通用）和局部（仅当前角色卡生效）两种作用域。

### 宏系统

| 宏 | 说明 |
|---|---|
| `{{char}}` | 当前角色名 |
| `{{user}}` | 用户名 |
| `{{lore_entries}}` | 待分析条目列表 |
| `{{lore_context}}` | 聊天正文 |
| `{{lore_constraints}}` | 约束规则 |
| `{{lore_user_persona}}` | 用户人设 |
| `{{条目别名}}` | 条目内容（需在约束中设置 macro 字段） |

### 正文提取

- **include 标签**：只提取指定 XML 标签内的内容
- **exclude 标签**：排除指定标签（`*` = 排除所有标签）
- 在 AI 配置页可实时预览提取效果

### 条目过滤

- **全部**：分析所有条目（默认）
- **仅处理指定**：只分析勾选的条目
- **排除指定**：分析除勾选外的全部条目

### 备份与回档

- 每次 AI 分析前自动备份世界书快照
- 可在世界书页签中查看备份列表并一键回档
- 支持设置备份间隔和最大保留数

## 🧪 调试

在 **调试面板** 中可以：

- 运行系统自检（API 连通性、世界书状态）
- 预览管线提示词（查看发送给 AI 的完整内容和 Token 统计）
- 查看 AI 调用历史
- 使用测试模式模拟写入（不消耗 API 额度）
- 导出完整调试日志

### 项目结构

```

├── index.ts                   # 插件入口
├── settings.ts                # 设置 Schema + Pinia Store
├── state.ts                   # 运行时状态
├── logger.ts                  # 日志系统
├── core/
│   ├── ai-engine.ts           # AI 调用 + 提示词组装
│   ├── update-pipeline.ts     # 更新管线（全流程串联）
│   ├── constraints.ts         # 约束 + 宏系统
│   ├── worldbook-api.ts       # 世界书 CRUD API
│   ├── worldbook-context.ts   # 世界书上下文探测
│   ├── context-extractor.ts   # 正文提取引擎
│   ├── persona.ts             # 用户人设读写
│   ├── scan-cache.ts          # SCAN_DONE 缓存管理
│   ├── prompt-editor.ts       # 提示词编辑器（DOM）
│   ├── review-editor.ts       # 审核弹窗（DOM + LCS Diff）
│   ├── backup-manager.ts      # 自动备份
│   └── test-mode.ts           # 测试模式
├── stores/
│   └── selfCheckStore.ts      # 自检状态
├── styles/
│   └── lorevnter.scss         # iOS 风格设计系统
└── window/
    ├── LorevnterWindow.vue     # 主窗口
    ├── components/
    │   └── ContextBar.vue      # 状态栏
    └── tabs/                   # 六个功能页
```

### 技术栈

- **框架**: Vue 3 + Pinia
- **构建**: Webpack 5 + TypeScript
- **UI**: iOS 风格（圆角卡片、柔和阴影、分组列表）
- **运行环境**: SillyTavern 酒馆助手脚本
